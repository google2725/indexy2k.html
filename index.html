Welcome to Incognito (INC)

Incognito is a privacy-focused Layer 1 blockchain built from scratch using the CryptoNote protocol. Our mission is to provide truly private, untraceable transactions for everyone.

Key Features

  • Ring Signatures - Hide the true sender among a group of possible signers
  • Stealth Addresses - One-time addresses for each transaction

About CryptoNote

CryptoNote is an open-source technology that allows the creation of privacy-focused cryptocurrencies. Key innovations include untraceable payments via ring signatures, unlinkable transactions via one-time keys, double-spending proof, blockchain analysis resistance, and egalitarian proof of work.
